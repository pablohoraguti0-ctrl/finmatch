# FinMatch shared components

Os componentes globais da experiencia multi-pagina sao renderizados por `assets/js/global.js` e estilizados por `assets/css/global.css`.

- Navbar global com mega menu, busca e conta demo.
- Footer global com links de plataforma.
- Home platform layer: onboarding, dashboard, comparador, score, favoritos, alertas, jornada, glossario e newsletter.
- Mobile bottom navigation.
- Cross-sell automatico nas paginas internas.

Essa abordagem evita repetir HTML em todas as paginas e mantem a identidade visual centralizada.
